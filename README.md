# Fake News Detection Web Application

This project is a Machine Learning-based application that classifies news articles as real or fake.

## Overview
The model is trained on a large dataset of news articles and uses Natural Language Processing (NLP) techniques to analyze and classify text.

## Features
- Text preprocessing (tokenization, stopword removal, TF-IDF)
- Logistic Regression model
- Real-time prediction capability

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy

## Dataset
- Used a dataset of 30,000+ news articles
- Balanced dataset for improved model performance

## Results
- Achieved ~92% accuracy on classification task

## How to Run
1. Clone the repository
2. Install required libraries:
   pip install pandas scikit-learn numpy
3. Run the model:
   python train_model.py
